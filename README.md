说明：
=============
此为带注释版的可运行手机上的GitHub,
源自于开源项目：[android](https://github.com/github/android)<br>
由于很少有注释，很难阅读，为了学习需要，由此延展出一份带解读的版本



项目导航
=============
* com.github.mobile
  <br>运行在手机上的GitHub
* ActionBarSherlock
  <br>GitHub运行所需的三方库
* com.viewpagerindicator
  <br>GitHub运行所需的三方库
